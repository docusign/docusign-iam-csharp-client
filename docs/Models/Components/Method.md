# Method

The HTTP method used to trigger the workflow. This defines the type of request
that will initiate the workflow (e.g., GET, POST).


## Example Usage

```csharp
using Docusign.IAM.SDK.Models.Components;

var value = Method.Get;
```


## Values

| Name     | Value    |
| -------- | -------- |
| `Get`    | GET      |
| `Post`   | POST     |
| `Put`    | PUT      |
| `Patch`  | PATCH    |
| `Delete` | DELETE   |
